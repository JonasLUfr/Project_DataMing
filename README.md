# 亚马逊电商销售数据分析
## 一、项目摘要（Abstract）
本项目基于真实的亚马逊销售数据，旨在通过数据清洗、可视化、探索性分析及聚类建模，识别不同类型的商品销售特征及消费者行为规律。  
项目综合运用了 Python、Pandas、Scikit-learn、Matplotlib、Seaborn 等工具，对数据进行清洗、标准化、降维（PCA），并分别采用 K-Means 与 高斯混合模型（GMM），DBSCAN 进行聚类分析等机器学习技术。  
最终结果揭示了如「高折扣高销量」、「高价高评分」等关键群体特征，为企业制定定价、促销及推荐策略系统提供了相应的数据支持。  

## 二、项目背景与目标（Introduction）
随着电商数据规模的急剧增长，如何从大量商品与销售信息中提取有价值的商业洞察，成为企业核心竞争力的关键。亚马逊作为全球最大的电商平台，其海量商品数据为数据挖掘与商业智能提供了理想的研究场景。
本项目的主要目标包括：  
  1.探索销售数据结构与分布特征；  
  2.识别商品群体特征与潜在市场规律；  
  3.利用聚类等机器学习算法进行市场细分与用户建模；  
  4.提出基于分析结果的商业优化建议。  
整个项目以数据科学完整流程为指导，从数据获取、预处理、建模到结果解释与可视化，形成一套可复用的分析框架。  

## 三、数据集介绍
3.1 数据来源
数据集来自 Amazon 官方产品销售公开数据集，部分字段经过匿名化与预筛选。
